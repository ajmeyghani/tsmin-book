# Angular and TypeScript

It is much easier to write Angular with TypeScript.

## Sample

A sample ts class with decoration:

```typescript
import {bootstrap} from 'angular';
@component({
  selector: 'app'
});
class App {}
```

The corresponding html

```html
<!DOCTYPE html>
<html>
<head>
  <title>example</title>
</head>
<body>
  <app [prop]='data'></app>
</body>
</html>
```

The corresponding css:

```css
.app {
  display: block;
}
```
